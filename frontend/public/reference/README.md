# Landing page reference assets

Source PNG artwork for the wedding landing page. These files are **not** served by the app; production uses optimized WebP under `frontend/src/assets/landing/`.

| File | Description |
|------|-------------|
| `RSVP.png` | RSVP section background (1080×960) |
| `RSVP_Date.png` | Date section artwork (1080×677) |
| `RSVP_Venue.png` | Venue section background (1080×661) |
| `RSVP_Site_Full V1.png` | Full-page design comp (1080×3678) |
| `couple-animation/RSVP_Site_GIF.png` | Couple sprite sheet, high-resolution export (9720×1080, 9 frames) |
| `couple-animation/RSVP_Site_GIF_landing.png` | Couple sprite sheet used for the landing build before WebP (9720×1080, 9 frames) |

Mirrors of the section PNGs and landing sprite also live in `frontend/src/assets/landing/originals/`.
